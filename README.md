Atividade de visualização de dados com matplotlib:

Atividade com utilização da biblioteca matplotlib do python. Podendo ser instalada com o comando:
"pip install matplotlib"

Servindo para mostrar a eficiência na utilização de gráficos para representar certos dados em análise de estoque.

O script gera quatro gráficos, cada um projetado para um tipo específico de análise de estoque:

1. Gráficos de linhas (Tendência de estoque diário):
    Monitorar a evolução da quantidade de um produto específico ao longo de 7 dias.

2. Gráfico de barras(Comparação de produtos):
    Comparar a quantidade atual em estoque de diferentes produtos.

3. Gráfico de pizza(Proporção de categorias):
    Visualizar a proporção do valor total do estoque distribuída por diferentes categorias de produtos.

4. Gráfico de dispersão(Preço vs Quantidade):
    Analisar a correlação (ou a falta dela) entre o preço unitário de itens e suas respectivas quantidades em estoque.
